Finds patterns in a list of strings or paths

Please see
https://bitbucket.org/path-pattern-finder/path-pattern-finder/src/master/README.md

To run
* on Windows, execute bin/path-pattern-finder.exe (or add the directory to your path)
* on Linux, execute "bash bin/path-pattern-finder" or similar
* as JARs, make sure the path-pattern-finder.jar and the jars in lib/ are part of your class-path and run the class:
com.owenfeehan.pathpatternfinder.PathPatternFinder

Note that it is primarily designed to be used as a Java library. The command line application implements only basic functionality.
