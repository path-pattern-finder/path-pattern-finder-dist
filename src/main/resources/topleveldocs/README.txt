Finds patterns in a list of strings or paths

Please see
https://path-pattern-finder.github.io/

To run
* on Windows, execute:
bin/path-pattern-finder.exe

* on Linux, execute:
bash bin/path-pattern-finder

* as Java, ensure path-pattern-finder.jar and the jars in lib/ are part of your class-path and run the class:
com.owenfeehan.pathpatternfinder.PathPatternFinder

It is recommended to add the bin/ directory to the system path.

Note that it is primarily designed to be used as a Java library. The command line application implements only basic functionality.
